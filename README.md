# Color Generator

A simple and interactive random color generator built using HTML, CSS, and JavaScript. Click the button to generate a random color and instantly copy its HEX code.

## Live Preview 
Open the `index.html` file in your browser — no setup required.

## Features
- Generates random HEX color codes  
- Button dynamically changes to the generated color  
- One-click copy of color code  
- Clean and minimal UI  
- Lightweight and beginner-friendly project  

## Built With
- HTML5  
- CSS3  
- Vanilla JavaScript  

## Project Structure
├── index.html
└── README.md

## How It Works
1. Click the Generate button  
2. A random color code is created  
3. The button background updates to the new color  
4. Click the color code to copy it  

## Example Output
#384920
#902134
#001293

## Notes
- The generator currently uses numeric HEX values only (0–9)  
- Can be extended to include A–F for full HEX color support  

## Possible Improvements
- Add full HEX characters (A–F)  
- Add RGB or HSL modes  
- Improve mobile responsiveness  
- Add copy confirmation feedback
